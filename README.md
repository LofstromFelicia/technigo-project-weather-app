# 🌤️ Väderappen | Live Weather App

A modern, sleek, and personalized weather application built with a focus on a **Glasmorphism UI** (frosted glass aesthetic) and user-friendly, dynamic clothing recommendations based on current weather conditions. (An idea based on the Swedish saying: There is no bad weather, only bad clothing)  
The app fetches real-time data and forecasts using the OpenWeatherMap API.

(./assets/sol-och-moln.svg) *Is it stubbornly raining in Tierp again? The app will display the true, cozy rain cloud to match!*

## ✨ Features

* **Real-Time Weather:** Displays current temperature, weather descriptions, wind speed, humidity, and precipitation probability for the searched city.
* **Clothing Recommendations ("Kläder efter väder"):** A smart, custom recommendation panel providing charming, dynamic outfit tips based on temperature, weather codes, and whether it is day or night.
* **3 & 6-Hour Forecast:** Displays a quick short-term forecast for the upcoming hours with matching custom icons.
* **5-Day Forecast:** Provides a clean and structured breakdown for the upcoming days.
* **UV Index with Warnings:** Dynamically calculates the UV index and shows clear alerts when sun protection or shade is recommended.
* **Smart Sun Times:** Displays exact, localized times for sunrise and sunset.
* **Multi-Language Screen Hack:** Hardcoded fixes to ensure compatibility with automatic browser translation tools without breaking the layout (e.g., resolving the classic "Ons/US" abbreviation translation glitch for Wednesdays!).

## 🛠️ Tech Stack

* **HTML5** – Semantic structure.
* **CSS3** – Custom layout utilizing Flexbox, CSS variables, keyframe animations, and advanced `backdrop-filter` properties for the glasmorphism effect.
* **JavaScript (ES6+)** – Asynchronous programming (`Fetch API`), dynamic DOM manipulation, and logical data processing.
* **API** – OpenWeatherMap (Current Weather + 5 Day / 3 Hour Forecast).
